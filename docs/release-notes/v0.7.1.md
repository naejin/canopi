# Canopi 0.7.1 Release Notes

Release date: 2026-06-16

## Highlights

- Added flat typed Object Groups so group members are now persisted by concrete kind (`plant`, `zone`, `annotation`) instead of a single anonymous ID list.
- Enabled grouping from the Action Toolbar across mixed canvas object types, including adding new objects into existing groups.
- Added support for merging existing groups during Group, including preserving a target group's identity and removing superseded groups.
- Carried group membership typing through reuse flows (copy/paste, duplicates, deletions, stamps, selection transforms, and locks) so grouped content stays consistent.

## Fixes And Refinements

- Added migration support for legacy `member_ids` groups, with explicit diagnostics for ambiguous/missing members.
- Updated group selection/interaction behavior to derive lock, visibility, and hit-testing from concrete member targets.
- Added regression coverage for mixed-type grouping, existing-group merges, and typed migration edges.
