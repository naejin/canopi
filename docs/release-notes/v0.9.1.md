# Canopi 0.9.1 Release Notes

Release date: 2026-06-25

## Highlights

- Refreshed the bundled plant database from `canopi-export-2026-06-25.db`.
- Pinned Plant Names now stay at their default per-plant position, even when labels overlap.
- Multi-plant selections no longer show transient plant names, reducing misleading label placement on dense canvases.

## Fixes And Refinements

- Let the pinned-name legend grow when canvas height is available before it falls back to scrolling.
- Removed steady-state location coordinate text from the canvas, while keeping loading, error, terrain, and precision notices.
- Changed the Welcome Screen title bar to show `Canopi`; new designs still start with editable `Untitled Design`.
