# Canopi 0.9.2 Release Notes

Release date: 2026-07-08

## Highlights

- Added the Web Edition packaging path, including the browser app shell, static design-template imports, reduced species catalog assets, and release artifacts for publishing Canopi under `/app/`.
- Added the Design Notebook panel with sections, pinned/current/recent design affordances, persisted ordering, and simplified navigation.
- Added Design Report PDF export with canvas, timeline, budget, consortium, and localized report chrome.
- Updated the plant database pipeline for canopi-data export schema v14, reducing the prepared database by omitting unused export tables and replacing source-based Common Name ranking with `display_order`.

## Fixes And Refinements

- Improved species catalog search responsiveness with off-main-invoke execution, cancellation for stale searches, indexed active-search entries, and better relevance ranking.
- Removed obsolete plant detail provenance fields, image source badges, and the Related Species section that no longer belongs to the app contract.
- Fixed Linux startup freezes caused by the WebKitGTK DMABuf renderer path.
- Restored location panel stability when opening designs with saved locations and removed the now-unused map compass.
- Cleaned up obsolete shell chrome, dormant site-adaptation code, learning stubs, saved-stamp prototype entrypoints, and stale panel location seams.
