# Canopi Desktop 1.0.0 Release Notes

Release date: 2026-07-09

## Highlights

- First 1.0 desktop release for Canopi's agroecological garden design workflow, packaging the app for Linux, macOS Apple Silicon, macOS Intel, and Windows.
- Includes the field-notebook desktop shell with canvas drawing tools, plant placement, layers, Design Notebook, Species Catalog, Favorites, Location, Timeline, Budget, and Consortium panels.
- Ships the current `.canopi` document lifecycle with save/load, document switching, dirty-state handling, and persisted non-canvas design data.
- Bundles the v14 plant database contract and the latest species search improvements from the 0.9.x release line.
- Keeps the Linux WebKitGTK renderer startup fix so saved-location designs can open without requiring a shell workaround.

## Since 0.9.2

- Added root-subdomain Web Edition artifacts and stricter Web artifact verification in the shared release tooling.
- Improved Web Edition species placement flows, including the catalog Place action, mobile filter collapse behavior, tablet filter coverage, and selected labels after plant-stamp placement.
- Recorded the production Web Edition deployment fixes that share this app version.
