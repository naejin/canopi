# Canopi 0.9.0 Release Notes

Release date: 2026-06-24

## Highlights

- Added canvas measurement guides for planning distances, including guide creation, selection after creation, editing, and aligned guide labels.
- Added pinned plant names on the canvas, with a dedicated legend and refreshed pin/unpin action state for plant name markers.
- Improved consortium entry layout by packing derived lanes so overlapping entries remain easier to scan.

## Fixes And Refinements

- Updated French saved-stamp actions to consistently use `planche`.
- Kept newly created measurement guides selected so they can be adjusted immediately.
