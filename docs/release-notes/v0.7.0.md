# Canopi 0.7.0 Release Notes

Release date: 2026-06-15

## Highlights

- Added inline annotation editing in the canvas with refined focus and shortcut behavior.
- Completed the plant symbol workflow: species defaults, persistent symbol IDs, symbol rendering, and applying symbols across selected items.
- Added plant drag spacing guides to make spacing and drag-distance feedback clearer.
- Routed map layer actions through design-location workflows for more consistent map and location interactions.
- Raised the canvas precision zoom cap and improved toolbar placement/handling for selection and rotation interactions.

## Fixes And Refinements

- Fixed context-menu and clipboard edge cases that could mutate blocked or invalid targets.
- Prevented selection toolbar and overlays from interfering with drag and rotation gestures.
- Fixed invalid plant symbol drag handling and improved symbol preview sizing and popover presentation.
- Corrected spacing and paste behavior for context actions, including anchor and target selection rules.
- Added regression coverage for the canvas UX and symbol/spacing flows.
