# Canopi 0.8.0 Release Notes

Release date: 2026-06-21

## Highlights

- Added Saved Object Stamps for reusable garden arrangements, including saving from the current canvas selection, placing stamps back onto the canvas, drag-to-place, import, export, inline rename, delete confirmation, manual ordering, and persisted split height in the Favorites panel.
- Refined the Saved Stamps UI into the existing field-notebook Favorites panel with compact rows, larger stamp names, spatial thumbnail previews, and French `Planches` copy for the saved arrangement library.
- Added richer plant symbol controls with a two-row picker, new groundcover and wave symbols, and improved shrub/groundcover canvas glyphs.
- Added editable Design names from the title bar and an About Canopi dialog from the Help menu.
- Added zone control point editing for more direct zone-shape refinement on the canvas.

## Fixes And Refinements

- Made Saved Stamp reordering feel live and predictable: the preview updates during pointer drag, can move below the bottom row, survives downward row reflow, commits the previewed order, and responds sooner in the drag direction.
- Hardened Saved Stamp portability and preview fidelity so zones, annotations, and plant arrangements remain recognizable when reused.
- Improved plant placement focus and canvas activation after plant drops, including returning to Select and limiting drag-distance guides.
- Stabilized ellipse boundary hit testing, zone handle behavior, annotation double-click state, and toolbar tooltip handling.
- Updated favorite plant rows to prioritize climate zone and life cycle for planning context.
