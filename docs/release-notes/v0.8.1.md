# Canopi 0.8.1 Release Notes

Release date: 2026-06-21

## Fixes And Refinements

- Fixed the About Canopi dialog so it reports the app release version from `desktop/tauri.conf.json`, showing `0.8.1` for this release instead of stale build metadata.
- Tightened release-candidate preflight checks so Rust workspace, Tauri, npm package, and npm lockfile versions must stay synchronized before packaged artifacts are built.
- Documented release-note promotion behavior so prepared `docs/release-notes/v<version>.md` files are used as the base GitHub Release body.
