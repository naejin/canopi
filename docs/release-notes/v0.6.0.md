# Canopi 0.6.0 Release Notes

Release date: 2026-06-03

## Highlights

- Added local Problem Reports with diagnostic bundle generation, redacted logs, frontend diagnostics, environment metadata, and an explicit-consent path for attaching the current `.canopi` design.
- Added rectangular, elliptical, and polygonal zone drawing improvements, including measurement overlays, snapped geometry, and safer polygon draft interactions.
- Added the Object Stamp flow for repeating plants, zones, annotations, and grouped canvas content.
- Added the Plant Spacing tool with a dedicated HUD, preview capping, source revalidation, tighter drag/guide behavior, and stronger keyboard and focus handling.
- Added canvas toolbar undo and redo controls.
- Remembered more workspace layout preferences, including species catalog sidebar width and bottom panel heights per tab.

## Fixes And Refinements

- Improved species catalog filtering on narrow layouts with better wrapping, proportional sidebar defaults, capped filter regions, and reserved row height for stable interaction.
- Hardened location and map surfaces, including startup failure handling and canvas notices that stay clear of application chrome.
- Tightened PanelBar button tooltip behavior and timeline hover handling while panning.
- Deepened frontend interaction seams across canvas, timeline, location, budget, design session, and species catalog flows so scene edits route through clearer runtime boundaries.
- Updated agent-facing documentation for document lifecycle, frontend patterns, canvas runtime, database, MapLibre, problem reporting, and release polling.
